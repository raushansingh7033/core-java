# Operator
- Operator is a symbol which tells the compiler to perform some operations
- operator is special symbol which is used to perform an operarion between two operand

# types of operator
1. Arithmetic operator(+,-,*,/,%)
2. assignment operator
- simple assigment operator
a=10;
- chained assignment operator
a=b=c=10
- compound assignment operator
a*=5;
3. bitwise operator(&,|,^)
4. bitwise complement operator(~)
5. conditional operator(?:)
6. instanceOf Operator
7. increment and decrement opertor
8. short circuit operator(&&,||)
9. string concatenation operator
10. realtional opeartor
11. equality operator
